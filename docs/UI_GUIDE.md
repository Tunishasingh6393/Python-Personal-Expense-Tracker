# Detailed FinTrack UI Walkthrough

This document contains high-resolution layout mappings and UI component descriptions.

## Primary Dashboard Components
1. **Nav Sidebar**: Integrated project mode switch (Synthetic vs Live).
2. **KPI Matrix**: High-contrast typography for Balance, Income, and Outflow.
3. **Allocation Area**: Dynamic Recharts pie chart with active-state highlighting.
4. **Historical Grid**: Dense, sortable ledger for forensic accounting.
